# ML Quant Trading System

A machine learning–driven quantitative trading research pipeline built in Python.
This project demonstrates the complete workflow used in quantitative finance: collecting market data, engineering predictive features, training machine learning models, generating trading signals, and evaluating strategies through backtesting.

The system uses historical stock market data to build predictive models that attempt to forecast future price movements and simulate trading strategies based on those predictions.

## Features

• Automated historical market data collection
• Financial time-series analysis
• Feature engineering (returns, momentum, volatility indicators)
• Machine learning model training using scikit-learn
• Signal generation for buy/sell decisions
• Strategy backtesting and performance evaluation
• Visualization of market data and results

## Tech Stack

Python
pandas
yfinance
scikit-learn
matplotlib

## Project Goal

The goal of this project is to understand and implement the core components of a quantitative trading system while applying machine learning techniques to financial time-series data.

## Learning Outcomes

• Understanding financial market data (OHLCV time series)
• Building machine learning pipelines for prediction tasks
• Developing trading signals from model outputs
• Evaluating trading strategies using backtesting methods
